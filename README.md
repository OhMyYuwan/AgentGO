# Agent GO!

![Agent GO!](./assets/github-header-banner.png)

## 🌐 教程页面

点击打开：[Agent GO! 教程浏览器](https://ohmyyuwan.github.io/AgentGO/)

本地预览可以在项目根目录启动静态服务：

```shell
python3 -m http.server 8080
```

然后访问 `http://localhost:8080/web/index.html`。
